## Description
TTY text editor like nano. Action saving, cut, copy, paste, undo, redo. It should be able to open a 1 GB file. It should have a setting so that we can autosave with delay. The search should be there

## Setup 

### Using .nvmrc file
```
nvm use
```

### Installation
```
npm install
```

### Running the app

```
# run typescript server
npm run ts-server

# run the application
npm run dev filename
```
